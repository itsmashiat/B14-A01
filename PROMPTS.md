# AI Prompts Used for Custom Section
AI prompts used to generate and style the custom AI challenge section


## Prompt 1: Ideation and Structure

Act as a Product Designer and Frontend Developer. 
I am building a landing page for 'DevConf 2026', a premium developer conference. I need a brand-new, unique section that fits the theme of the conference and provides additional value to attendees.
Suggest a creative section idea and write the clean semantic HTML5 layout for it.
Restrictions:
- Pure HTML, no JavaScript.
- Must include a clear heading, supportive descriptions, a grid of details.
- Clean class names suitable for modern CSS styling.


## Prompt 2: Modern Dark Tech CSS Styling

Write a CSS design system and styling for the Hackathon Arena section.
Aesthetics must be extremely premium:
- Use glassmorphism
- No JS. All hover states and card details must be animated smoothly with CSS transition properties.
